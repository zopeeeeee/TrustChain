    1. Create backend/app/services/detection.py:
       ```python
       async def run_detection(upload_id, file_path, models, session):
       ```
       - Takes upload_id (UUID), file_path (str), models (dict from app.state), session (AsyncSession)
       - Derives job_dir from file_path parent, frames_dir = job_dir / "frames", audio_path = job_dir / "audio.wav"
       - Stage 1 - Visual: update status to "visual_analysis", run extract_visual_features in thread pool via asyncio.to_thread (CRITICAL: prevents event loop blocking)
       - Stage 2 - Audio: update status to "audio_analysis", call detect_speech. If speech detected AND audio.wav exists, run extract_audio_features via asyncio.to_thread, set audio_weight=1.0. Otherwise, audio_features = torch.zeros(768), audio_weight=0.0
       - Stage 3 - Fusion: update status to "computing_fusion", run fusion_model forward via asyncio.to_thread. confidence = fusion result (float). verdict = "FAKE" if confidence > 0.5 else "REAL". visual_score = confidence (for prototype, or use L2 norm of visual features as a simple score). audio_score = confidence if has_speech else 0.0.
       - Stage 4 - Store: update upload with verdict, confidence, visual_score, audio_score, speech_detected (bool), audio_weight, completed_at=datetime.utcnow(), status="completed"
       - Wrap everything in try/except: on failure, set status="failed", error_message=str(e)
       - Use update_upload_status from preprocessing.py for status updates, or create a similar helper in this module that also sets result fields
