  <action>
    1. Add `numpy` and `Pillow` to backend/requirements.txt (needed for PREP-01 frame normalization prep, though actual normalization will happen at inference time in Phase 3 -- just install now).

    2. Implement backend/app/services/preprocessing.py with these functions:

       `async def probe_video(video_path: str) -> dict`:
       - Run ffprobe via asyncio.create_subprocess_exec with -print_format json -show_format -show_streams
       - Parse stdout as JSON, raise RuntimeError on non-zero exit
       - Return the parsed dict

       `async def validate_video(video_path: str) -> tuple[bool, str]`:
       - Call probe_video, extract duration from format.duration
       - If duration < 1.0: return (False, "Video must be at least 1 second long")
       - On RuntimeError/KeyError/ValueError: return (False, "File appears to be corrupted or not a valid video")
       - Otherwise return (True, "")

       `async def extract_frames(video_path: str, output_dir: str, frame_interval: int = 10) -> list[str]`:
       - Run FFmpeg: ffmpeg -i {video_path} -vf "select=not(mod(n\\,{frame_interval})),scale=224:224" -vsync vfr -q:v 2 {output_dir}/frame_%04d.jpg -y
       - Use asyncio.create_subprocess_exec (NOT shell=True) -- escape the backslash in mod expression
       - On non-zero exit: raise RuntimeError with stderr
       - Return sorted list of frame file paths from output_dir using glob

       `async def extract_audio(video_path: str, output_path: str) -> bool`:
       - Run FFmpeg: ffmpeg -i {video_path} -vn -acodec pcm_s16le -ar 16000 -ac 1 {output_path} -y
       - On non-zero exit: check stderr for "does not contain any stream" or "no audio" -- return False (no audio track, graceful)
       - On other non-zero: raise RuntimeError
       - Return True on success

       `async def process_video(upload_id: uuid.UUID, file_path: str)`:
       - Create a NEW async_session (not the request's session -- critical pitfall)
       - Wrap everything in try/except
       - Step 1: validate_video -- if invalid, set status="failed", error_message=reason, return
       - Step 2: Update status="extracting_frames", extract frames to /data/{upload_id}/frames/
       - Step 3: Update status="extracting_audio", extract audio to /data/{upload_id}/audio.wav
         - If extract_audio returns False: set audio_weight=0.0 on Upload record
       - Step 4: Update status="queued" (ready for Phase 3 ML analysis)
       - On any exception: set status="failed", error_message=str(e)
       - Each status update: fetch Upload by ID, update field, commit

       Helper `async def update_upload_status(session, upload_id, status, **kwargs)`:
       - Fetch upload, set status and any kwargs, commit
